Description:
A console-based Student Management System built in Java that allows administrators to manage student records efficiently. 
Each student is assigned a unique ID and stores details like name, age, course, and email. 
The system supports full CRUD operations — add, view, update, and delete students.
Additional features include subject-wise grade tracking, keyword-based search across name, course, and email, and filtering students by course name.

Conclusion:
This project demonstrates core Java concepts including HashMap for record management, ArrayList for grade storage, and structured menu-driven input handling. 
It covers real-world use cases like searching, filtering, and updating records — making it a solid project for understanding data management in Java without using a database.
Suitable for beginners looking to practice Java OOP and collections.
